Implementation

The Pear pixel is a snippet of JavaScript code that enables your e-commerce website to receive and optimize high-intention shopper traffic from Pear-enabled traffic sources such as CPG ads, brand websites, social media influencers, and mass-media agencies. It relies on several web technologies, including user opt-in cookies, which enable deep-linking shoppers into your shopping cart and re-engaging with them to bring them back to your stores.

The pixel has been designed from the ground up with security and compliance in mind. It is only made aware of shoppers who arrive directly from a Pear-enabled traffic source, and only uses the bear minimum amount of data to provide a superior consumer experience. In addition, the functionality of the pixel is fully modular, so we are happy to work with you if you have specific needs or requests.

One-line Installation

The simplest approach to installing the pixel is by including the following line of code in your shopping cart website:

<!-- Pear Pixel Snippet -->
<script type="application/javascript"
 src="https://offers.pearcommerce.com/js/sdk.js">
</script>
